Introdução ao HTML

HTML significa Hyper Text Markup Language

HTML é a linguagem de marcação padrão para a criação de páginas da web

HTML descreve a estrutura de uma página da web

HTML consiste em uma série de elementos

Atributo href

Os links HTML são definidos com a tag a. O endereço do link é especificado no atributo href

Visita w3schools

O atributo src

As imagens HTML são definidas com a tag img, e o nome do arquivo da fonte da imagem é especificado no atributo src.
