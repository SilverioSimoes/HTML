Introdução ao HTML

HTML significa Hyper Text Markup Language

HTML é a linguagem de marcação padrão para a criação de páginas da web

HTML descreve a estrutura de uma página da web

HTML consiste em uma série de elementos

Atributos HTML

Atributo href

Os links HTML são definidos com a tag a. O endereço do link é especificado no atributo href

Visita w3schools

O atributo src

As imagens HTML são definidas com a tag img, e o nome do arquivo da fonte da imagem é especificado no atributo src.

O atributo alt

O atributo alt deve refletir o conteúdo da imagem, para que os usuários que não podem ver a imagem entendam o que a imagem contém.

O atributo alt obrigatório para a img tag especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida. Isso pode ser devido a uma conexão lenta, um erro no src atributo ou se o usuário usa um leitor de tela.

Atributos width e height

Os atributos weidth e heigth da tag img definem a largura e a altura da imagem.

O atributo style

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Este parágrafo é vermelho.

O atributo title

Passe o mouse sobre este parágrafo para exibir o atributo title como um tooltip.

O atributo lang

Você deve sempre incluir o atributo lang dentro da tag html, para declarar o idioma da página web. Isso serve para auxiliar os mecanismos de pesquisa e navegadores.

Os códigos de país também podem se adicionados ao código de idioma no atributo lang. Portanto, os primeiros dois caracteres definem o idioma da página HTML e os dois últimos caracteres definem o país.
